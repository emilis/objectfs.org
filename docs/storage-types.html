Support for different storage types is implemented in ObjectFS packages.

ObjectFS parses storage URIs that you provide and finds the appropriate package from this table1:

Matching package scheme: //host /path
objectfs-core
ObjectFS core library		 csv * *
* * *.csv
ofs-sqlite
ObjectFS storage wrapper for Sqlite		 sqlite * *
* * *.sqlite

1 You can change your local URI matching table. It is stored in ../objectfs/config/uri_map.csv.
