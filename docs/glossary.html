Term(s) Description
Storage

Anything that can be used as a set of records.

Examples: DB table, list of databases on server, CSV file, RSS feed, results from an API call, directory full of image files, etc..

Record

Anything that can be expressed as a JSON object (strings for keys and a few value types).

Example: {id:123,name:"Item 1"}.

URI / Storage URI

Universal Resource Identifier – a string that represents a single storage.

Example: mysql://example.org/database/table.

Wrapper / Storage wrapper

A JavaScript module that provides a standard interface for working with records in a storage.

ID / Record ID

A value (number/string/?) that identifies a single record within a storage.

Path

A string that identifies a part of a storage.
Example: /var/log/nginx/ -- just the nginx log files.

(ObjectFS) Package

A collection of modules, code and other assets conforming to CommonJS Package specification. You can install, update and remove packages using the ObjectFS package manager ofs-pkg.

Example: ofs-sqlite package.
