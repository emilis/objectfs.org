ObjectFS is a set of libraries and programs for accessing records in all kinds of storage engines and formats.

It functions by using a variety of storage drivers for different kinds of data services and formats.

ObjectFS architecture strives to make writing, sharing and connecting these drivers as easy as possible.

You can use ObjectFS as:

  • a program,
  • a library
  • or a scripting shell.
