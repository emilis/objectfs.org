--- title: Easy data plumbing layout: objectfs ---

ObjectFS is an open platform for moving data between all kinds of databases, file formats, APIs, etc.

A quick example: move all blog posts from WordPress to CouchDB:

$ ofs mirror wordpress://example.org/ couchdb://example.net/blog/
