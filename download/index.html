Requirements:

You will need:

  • Java 6 or later (7-25 MB download if you don't have it yet).
  • Git (to update packages).
    • A properly set-up GitHub account if you'll want to publish packages or send patches.

Stable version:

Download
build 2011-11-21, 3.7 MB

Installation

No installation steps are required. Just extract the archive and you can run scripts bin/ofs or bin/ofs-pkg.

See also: Keeping up to date.

Development version:

To get the latest version from GitHub run:

$ git clone git://github.com/emilis/objectfs.git
$ cd objectfs
$ make build

Keeping up to date

Nearly all of ObjectFS functionality is defined in JavaScript packages. You can install and remove these packages using ObjectFS package manager ofs-pkg.

It is advisable to update the package list often by running:

$ ofs-pkg upgrade

You will rarely need to update ObjectFS itself.
